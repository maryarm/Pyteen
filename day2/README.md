# 📚 Day 2 — Python Fundamentals

Day 2 covers the building blocks that every Python program is made of.
Each topic has its own file so you can learn and practice at your own pace.

---

## 📁 Files in This Folder

| File | Topic | What you will learn |
|------|-------|---------------------|
| `variables_and_types.py` | Variables & Data Types | How to store information in named boxes (`str`, `int`, `float`, `bool`) |
| `input_output.py` | Input & Output | How to show messages and collect answers from the user |
| `comments.py` | Comments | How to write notes inside code that Python ignores |
| `operators.py` | Basic Operators | How to do math and comparisons (`+`, `-`, `*`, `/`, `==`, `>`, etc.) |
| `if_else.py` | If / Else | How to make your program choose different paths |

Each `.py` file has a matching `.md` guide with activities and discussion questions.

---

## ▶️ Suggested Order

1. **variables_and_types** — you need variables before everything else
2. **input_output** — make programs that talk to the user
3. **comments** — good habits from day one
4. **operators** — the math and comparisons you will need for decisions
5. **if_else** — putting it all together with real choices

---

## 🏁 How to Run Any File

```bash
python variables_and_types.py
python input_output.py
python comments.py
python operators.py
python if_else.py
```

---

## 🎯 Teaching Philosophy

Every file in Day 2 follows the same approach:

1. **Read first** — students read working code before writing anything
2. **Tweak to learn** — small, safe changes with instant visible results
3. **Understand why** — comments explain ideas, not just syntax
4. **Build up gradually** — each activity goes one step further

Students with zero experience can run something real in the first minute.
