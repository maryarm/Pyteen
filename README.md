# 🐍 Pyteen

Python exercises and mini-projects for teenagers learning programming and algorithms for the first time.

Each project is self-contained, beginner-friendly, and comes with a guide for students and teachers.

---

## 📁 Project Structure

```
Pyteen/
└── day1/
    ├── time_weather.py      ← fetches live weather and local time for cities
    ├── time_weather.md      ← student & teacher guide for time_weather.py
    ├── career_compass.py    ← explore careers using sorting and filtering algorithms
    └── career_compass.md    ← student & teacher guide for career_compass.py
```

---

## 📦 Setup

**Requirements:** Python 3.8 or newer

```bash
pip install -r requirements.txt
```

---

## 🎯 Teaching Philosophy

Every project in Pyteen follows the same approach:

1. **Read first** — students read working code before writing anything
2. **Tweak to learn** — small, safe changes with instant visible results
3. **Understand why** — comments explain ideas, not just syntax
4. **Build up gradually** — each session's challenges go one step further

Projects are designed so a student with zero experience can run something real on day one.
